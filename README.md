Bike Sales Dashboard
📌 Project Overview

This project is an interactive Bike Sales Dashboard built in Microsoft Excel. It analyzes customer purchasing behavior using Pivot Tables, Pivot Charts, and Slicers, allowing users to explore sales trends and customer demographics dynamically.

🎯 Objectives
Analyze customer purchasing patterns.
Compare average income based on bike purchases.
Study customer commute distances.
Analyze purchasing behavior across different age groups.
Create an interactive dashboard for business insights.
